# 🚀 AI/ML Engineering Internship Tasks – DevelopersHub Corporation

## 📌 Overview

This repository contains my completed tasks for the **AI/ML Engineering Internship at DevelopersHub Corporation**.

The goal of these tasks is to build a strong foundation in:

* Data Analysis & Visualization
* Machine Learning (Regression & Classification)
* Time Series Prediction
* Real-world AI Applications

---

## 🧠 Tasks Completed

### ✅ Task 1: Iris Dataset – Data Exploration & Visualization

* Loaded dataset using `pandas` and `seaborn`
* Performed exploratory data analysis (EDA)
* Visualized:

  * Scatter plots (feature relationships)
  * Histograms (distribution)
  * Boxplots (outliers detection)

📊 **Key Skills:**

* Data inspection (`.info()`, `.describe()`)
* Data visualization using `matplotlib` & `seaborn`

---

### 📈 Task 2: Stock Price Prediction (Short-Term)

* Fetched real-time data using `yfinance`
* Predicted next-day closing price
* Applied **Linear Regression**

📊 **Key Skills:**

* Time series data handling
* API data fetching
* Regression modeling

📉 **Output:**

* Actual vs Predicted price graph
* Next-day stock prediction

---

### ❤️ Task 3: Heart Disease Prediction

* Used UCI Heart Disease dataset
* Built a **Logistic Regression model**
* Evaluated using:

  * Accuracy
  * Confusion Matrix
  * ROC Curve

📊 **Key Skills:**

* Binary classification
* Medical data interpretation
* Feature importance analysis

---

### 🏠 Task 6: House Price Prediction

* Built regression model to predict house prices
* Used features like:

  * Size
  * Bedrooms

📊 **Evaluation Metrics:**

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)

📊 **Key Skills:**

* Regression modeling
* Model evaluation
* Data preprocessing
